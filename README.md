# CHIP-8 Emulator

CHIP-8 emulator with graphics and sound, written in c++.  
## Special keys
`+`, `-` : change the execution speed.  
`Esc`    : closes the game window.

## Keyboard
Can be configured at [Chip8Emulator/Constants.h](Chip8Emulator/Constants.h), default is:
|     |     |     |     |
| - | - | - | - |
| 1 | 2 | 3 | 4 |
| Q | W | E | R |
| A | S | D | F |
| Z | X | C | V |
